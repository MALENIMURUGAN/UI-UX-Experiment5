# UI-UX-Experiment5

## Aim:
To present 2 different UX project scenarios (MindEase - Mental Wellness App and QuickRide - Smart Parking App) including problem statements, research methods with justification, and detailed user personas.

## Algorithm:
**Step 1:** Identify user pain points for each scenario based on common real-world problems
**Step 2:** Write 2 clear UX problem statements per scenario using the template:
"[User] needs a way to [achieve goal] because [obstacle], which results in [impact]"
**Step 3:** Identify the most suitable research methods for each scenario (surveys, interviews, usability testing, diary studies, etc.)
**Step 4:** Justify each research method choice based on user group and scenario requirements
**Step 5:** Create 2 detailed user personas per scenario with:

Demographics

Goals

Behaviors

Pain Points

Motivation
**Step 6:** Compile all content into a structured lab report with clear headings and tables
**Step 7:** Prepare GitHub repository with markdown files and attach PDF for submission
